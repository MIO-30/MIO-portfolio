▼COMMENT

自分のポートフォリオなので、自分の好きなデザインで好き勝手してしまおうと思いました。訓練校の修了日まで時間もなく、見切り発車でデザインをはじめたのでデザインカンプは諦めました。indexは「冒険の書」や「セーブデータ」を選ぶようなイメージです。他のwedデザイナーの方々のような派手なデザインはまだ勉強中の自分には難しいと感じたので、デザインはシンプルですが直感的にわかりやすく、見ていて気持ちのいい動きになるように頑張りました。

今あるコーディングの知識をフルに使って制作しました。とはいえ、まだまだ理解しきっていない部分がとても多いと、制作の中で実感しました。制作していく途中で、わからないことがあると都度調べていたので、とても勉強になったと思います。

javascriptは「1冊ですべて身につく JavaScript入門講座」の本を参考に勉強しました。javascriptはもちろんですが、cssの記述で「grid」を使用しているのを見つけました。今まで使ったことがなかったのですが、調べてみると完全に理解するにはまだ勉強が必要そうですが、サムネイルの一覧をつくるときなどにとても便利だと感じました。このページと、イラストのサムネイル一覧にgridを使用しています。

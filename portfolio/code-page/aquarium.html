▼COMMENT

vtuberサイトの制作がひと段落したところで次の作品作りにかかりたいと考えました。友人と江ノ島水族館に遊びに行く予定があったので、水族館のサイトを作ることにしました。

検索上位のサイトを参考にしていく中で、激しいアニメーションは少なく、動きはゆったりしていることが多いと気が付きました。中でも、波がゆれるようなアニメーションがどのサイトにも一か所はあったので、自分も真似をしてメイン画像に波のようなアニメーションを追加しました。波はphotoshopで制作したのですが、SVGにしたほうがよかったかもしれないと反省しています。

vtuberサイトの制作では、javascriptを使用してアニメーションを実装していましたが、こちらの水族館のサイトはHTMLとcssのみで作っています。また、メディアクエリの設定をする際に細かい記述のし直しをしなくてもよいようにcssを書くよう心掛けました。
